# 中国PMI数据采集指南

## 数据来源优先级

1. **国家统计局英文版** `stats.gov.cn/english/PressRelease/`
   - 每月1日左右发布上月PMI
   - 提供完整历史数据表格（Table 1: 制造业PMI及5个构成指数; Table 2: 相关指数含新出口/进口/购进价格/出厂价格/库存等; Table 3: 非制造业）
   - URL格式: `https://www.stats.gov.cn/english/PressRelease/{YYYY}{MM}/t{YYYYMMDD}_{ID}.html`
   - **优势**: 数据最完整、格式最规范、含12个月历史对比

2. **国家统计局中文版** `stats.gov.cn/zwfwck/sjfb/`
   - 同日发布，内容与英文版一致
   - 配套解读文章: `stats.gov.cn/sj/sjjd/` (首席统计师解读)

3. **财新PMI** `pmi.caixin.com`
   - 通常比官方PMI晚1-3天发布
   - 侧重中小企业和出口导向，与官方PMI形成互补
   - 财新制造业PMI由S&P Global编制

4. **机构解读源**
   - 中国物流与采购网: `chinawuliu.com.cn/xsyj/` (特约分析师张立群评论)
   - 中信证券/国泰海通/银河证券: 新浪财经/腾讯新闻转载
   - 财新网: `economy.caixin.com` (深度分析)

## 企业规模分项（必看）

- 大型企业PMI: 通常高于整体，反映国企/龙头景气
- 中型企业PMI: 敏感指标，波动大
- 小型企业PMI: 最脆弱，常低于荣枯线

**关键信号**: 当大企扩张但中小企收缩时，说明政策传导不畅或外需退潮

## 行业分项（重点标注）

- 高技术制造业PMI: 新质生产力风向标，连续扩张=结构升级
- 装备制造业PMI: 制造业升级指标
- 消费品行业PMI: 内需消费温度计
- 高耗能行业PMI: 传统动能拖累程度

## 非制造业PMI关键分项

- 服务业商务活动指数: 消费/出行/金融景气
- 建筑业商务活动指数: 基建+地产投资温度计
- 非制造业新订单指数: 需求端
